# 🛠️ Como rodar o projeto

Este repositório contém dois projetos separados:

- `Backend/`: API em **Python**.
- `Frontend/controle-estoque/`: Aplicação **React**.

---

## ⚙️ Backend (Python)

### Pré-requisitos
- Python 3.10+
- pip

### Passos

```bash
cd Backend
py .\app.py 
```

---

## 🖥️ Frontend (React)

### Pré-requisitos
- Node.js 18+
- npm

### Passos

```bash
cd Frontend/controle-estoque
npm install
npm run dev         
```
